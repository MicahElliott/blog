Live Office Dashboard with i3 and a NUC

In this post I’ll show you how to set up a live dashboard to display your company (or house) metrics. It’s as simple as writing a little i3 script to rotate through a set of open browsers. You’ll need to run X, i3, a fast browser. The required hardware is a TV and a NUC.
